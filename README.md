# Theme Portfolio Task

Live link : https://theme-portfolio-task.vercel.app/

## Project Overview
This is a **personal portfolio project** built with **Next.js**. The goal of this project is to showcase a portfolio theme with responsive design, smooth navigation, and modern UI components using TailwindCSS and DaisyUI. It includes sections for projects, services, and skills, making it ideal for developers or designers to present their work.

## Tech Stack
- **Frontend Framework:** [Next.js](https://nextjs.org/) (v16.1.5)
- **UI Styling:** [TailwindCSS](https://tailwindcss.com/) (v4.1.18) + [DaisyUI](https://daisyui.com/) (v5.5.14)
- **PostCSS:** v8.5.6
- **React:** v19.2.3
- **Linting:** ESLint (v9)

## Getting Started / Run Locally

Follow these steps to run the project locally:

1. **Clone the repository**
   ```bash
   git clone <your-repo-url>
   cd theme-portfolio-task

2. **Install dependencies**
npm install


3. **Run the development server**
npm run dev

3. **Build for production**
npm run build
npm start
